Description: Retrieve BIN data filtered by bank, country, or BIN number.

Use Case: Fetch BIN details for specific criteria.

Parameters:

Parameter Type Description Default
bank String Bank name (case-insensitive, e.g., Chase). None
country String Country code (e.g., BD, CA). None
bin String Specific BIN number (e.g., 434907). None
limit Integer Number of results (1-500). Optional. None

Example Request (Bank):

GET /api/bin?bank=chase&limit=10
