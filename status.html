Description: Retrieves BIN data based on country or bank issuer.

Use Case: Fetch BIN details for a specific country or bank.

Parameters:

Parameter Type Description Default
country String Country code (e.g., BD, US). None
bank String Bank name (e.g., PRIME BANK, LTD, case-insensitive). None
limit Integer Number of results (1-500). Optional. 100

Example Request:

GET /api/bin?country=BD&limit=10

Example Response:

{
  "status": "success",
  "data": [
    {
      "bin": "101017",
      "brand": "LOCAL BRAND",
      "category": "PROPRIETARY",
      "country_code": "BD",
      "country_code_alpha3": "BGD",
      "country_name": "BANGLADESH",
      "issuer": "PRIME BANK, LTD.",
      "phone": "+88029662776",
      "type": "DEBIT",
      "website": "https://www.primebank.com.bd"
    }
  ],
  "count": 1,
  "filtered_by": "country",
  "api_owner": "@ISmartCoder",
  "api_channel": "@TheSmartDev"
}

Notes: The limit parameter allows you to control the number of results returned. It is optional and defaults to 100 if not specified.
